# minishell

a bash shell subset.

features implemented:

- logical operators (&&, ||)
- pipes
- redirections (<, >, <<, >>)
- variable expansion and environment variables
- $?
- single and double quotes for character grouping
- - wildcard for current directory
- builtins:
  - export
  - echo (with -n)
  - env
  - exit
  - cd
  - echo
  - pwd
  - unset
